## About

Deploy Apps in One-Click throw a SHELL script. <br/>
Deploy popular applications on Enterprise Linux el7 & el9 with a single click.

## Clone
```
git clone https://github.com/YanarAssaf/one-click-app
```
- cd el9 or el7
- chmod +x apache.sh
- ./apache.sh
