## About One-click-app

Deploy Apps in One-Click throw a SHELL script. 
Deploy popular applications such as Apache, BIND, Nginx, Nagios on CentOS7 with a single click.

Feathers:
- Apache
- Nagios
- Docker
- BIND
- CSF
- HAProxy

## Clone

- git clone https://github.com/YanarAssaf/one-click-app
- chmod +x apache.sh
- ./apache.sh
